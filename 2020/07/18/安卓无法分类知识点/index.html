finish()、onDestory()、System.exit()

摘录自：博文

  • finish() 在 Activity 动作完成的时候或需要 Activity 关闭时，调用此方法。此方法只是将最上面的 Activity 移出了栈，并没有及时 onDestory()，资源也未及时释放。
  • onDestory() 系统销毁了 Activity 的实例在内存中占据的空间。
  • System.exit(0) 针对 Application，直接将进程结束。

Q：如何设置类头注释

A：打开 Settings 找到 Editor 下的 File and Code Templates，在 Includes 下新建一个 Class Header，在其内输入想要添加的类头注释模板，然后在 Files 下对应的文件引用 Class Header.java。

${PACKAGE_NAME} 创建新文件在其中的包的名称。
${USER} 当前用户系统登录名。
${DATE} 当前系统日期
${TIME} 当前系统时间
${YEAR} 当前年份
${MONTH} 当前月份
${MONTH_NAME_SHORT} 当前月份名称的前3个字母。示例：Jan, Feb等。（英文下简称）
${MONTH_NAME_FULL} 当前月份的全名。示例：January, February等。（英文下全程）
${DAY} 当月的当前日期
${DAY_NAME_SHORT} 当前日期名称的前3个字母。例如：Mon, Tue等。(英文下简称)
${DAY_NAME_FULL} 当前日期的全名。例如：Monday, Tuesday等。(英文下全称)
${HOUR} 当前小时
${MINUTE} 当前分钟
${PROJECT_NAME} 当前项目的名称
